# Bot Audio Analyzer

Bot Audio Analyzer — учебный проект на Go, цель которого — создать инструмент для анализа аудиофайлов с возможностью дальнейшей интеграции с Telegram-ботом.

На текущем этапе проект получает информацию об аудиофайле с помощью `ffprobe`, преобразует её в удобный для использования вид и выводит основные характеристики: имя файла, длительность, формат, кодек, битрейт, частоту дискретизации, количество каналов и другие параметры. Архитектура проекта постепенно разделяется на отдельные компоненты, отвечающие за взаимодействие с `ffprobe`, обработку данных и их форматирование.

Проект создаётся в образовательных целях и используется для изучения языка Go, работы с внешними процессами, обработки JSON, проектирования структуры приложения и построения расширяемой архитектуры. В дальнейшем планируется добавить анализ громкости (LUFS), BPM, тональности, генерацию спектрограмм и waveform, а также интеграцию с Telegram для автоматического анализа загружаемых аудиофайлов.


## Настройка окружения

Скопируйте файл с примером переменных окружения:

```bash
cp .env.example .env
```

Заполните `.env` своими значениями:

```env
TELEGRAM_BOT_TOKEN=your_telegram_bot_token
```

Токен Telegram-бота можно получить через **BotFather**.

## Как это работает

После запуска бот автоматически получает `chat_id` из входящих сообщений Telegram через Bot API (`getUpdates`).

Дополнительная настройка `TELEGRAM_CHAT_ID` больше не требуется.
